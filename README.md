# picforword
> 此作品通过对COCO的数据文件进行分析、总结、建立句子难度模型，最终形成一个以COCO数据为基础的用图片、句子和相应的图片标注区域来解释汉语基础词语的工具。汉语学习者可以用此系统搜索词语，输出与词语相关的句子、图片与之相应的图片标注区域。
> 汉语学习者可清晰地理解汉语词的意思与使用以及汉语词之间的关系。
体验地址：http://qinsw666.top/picforword/picforword.html
